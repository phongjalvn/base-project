## NPM Modules

Initital NPM with:

```
npm init
```

Now you have `packages.json` file, which will contain your node modules configs.

Note: **You must install packages with options:** `--save-dev`

- gulp
- del
- browser-sync
- gulp-autoprefixer
- gulp-cache
